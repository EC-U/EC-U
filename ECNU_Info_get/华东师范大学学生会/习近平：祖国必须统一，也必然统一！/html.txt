 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <span style="color: rgb(86, 86, 86);font-size: 16px;">
   海峡两岸分隔已届70年。台湾问题的产生和演变同近代以来中华民族命运休戚相关。
  </span>
  <br/>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <br/>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <strong>
   <span style="font-size: 16px;color: rgb(0, 122, 170);">
    70年来的两岸关系
   </span>
  </strong>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <br/>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <span style="color: rgb(86, 86, 86);font-size: 16px;">
   70年来，我们顺应两岸同胞共同愿望，推动打破两岸隔绝状态，实现全面直接双向“三通”。
  </span>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <br/>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <span style="color: rgb(86, 86, 86);font-size: 16px;">
   70年来，我们秉持求同存异精神，推动两岸双方在一个中国原则基础上达成“海峡两岸同属一个中国，共同努力谋求国家统一”的“九二共识”。
  </span>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <br/>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <span style="color: rgb(86, 86, 86);font-size: 16px;">
   70年来，我们把握两岸关系发展时代变化，提出和平解决台湾问题的政策主张和“一国两制”科学构想，确立了“和平统一、一国两制”基本方针，进而形成了坚持“一国两制”和推进祖国统一基本方略。
  </span>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <br/>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <span style="color: rgb(86, 86, 86);font-size: 16px;">
   70年来，我们高举和平、发展、合作、共赢的旗帜，在和平共处五项原则基础上发展同各国的友好合作，巩固国际社会坚持一个中国原则的格局。
  </span>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <br/>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <span style="color: rgb(86, 86, 86);font-size: 16px;">
   70年来，我们始终着眼于中华民族整体利益和长远利益，坚定维护国家主权和领土完整，团结全体中华儿女，坚决挫败各种制造“两个中国”、“一中一台”、“台湾独立”的图谋。
  </span>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <br/>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;">

 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <strong>
   <span style="font-size: 16px;color: rgb(0, 122, 170);">
    两个“无法改变”和两个“无法阻挡”
   </span>
  </strong>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;">
  <br/>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <span style="color: rgb(0, 122, 170);">
   <strong>
    <span style="font-size: 16px;">
     两岸关系发展历程证明：
    </span>
   </strong>
  </span>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <br/>
 </p>
 <blockquote style="white-space: normal;">
  <p style="margin-right: 8px;margin-left: 8px;line-height: 1.75em;text-indent: 2em;">
   <span style="color: rgb(86, 86, 86);font-size: 16px;">
    台湾是中国一部分、两岸同属一个中国的历史和法理事实，是任何人任何势力都无法改变的！
   </span>
  </p>
  <p style="margin-right: 8px;margin-left: 8px;line-height: 1.75em;text-indent: 2em;">
   <span style="color: rgb(86, 86, 86);font-size: 16px;">
    两岸同胞都是中国人，血浓于水、守望相助的天然感情和民族认同，是任何人任何势力都无法改变的！
   </span>
  </p>
  <p style="margin-right: 8px;margin-left: 8px;line-height: 1.75em;text-indent: 2em;">
   <span style="color: rgb(86, 86, 86);font-size: 16px;">
    台海形势走向和平稳定、两岸关系向前发展的时代潮流，是任何人任何势力都无法阻挡的！
   </span>
  </p>
  <p style="margin-right: 8px;margin-left: 8px;line-height: 1.75em;text-indent: 2em;">
   <span style="color: rgb(86, 86, 86);font-size: 16px;">
    国家强大、民族复兴、两岸统一的历史大势，更是任何人任何势力都无法阻挡的！
   </span>
  </p>
 </blockquote>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;">
  <br/>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <strong>
   <span style="font-size: 16px;color: rgb(0, 122, 170);">
    五点主张
   </span>
  </strong>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;">
  <br/>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <span style="color: rgb(86, 86, 86);font-size: 16px;">
   回顾历史，是为了启迪今天、昭示明天。祖国必须统一，也必然统一。这是70载两岸关系发展历程的历史定论，也是新时代中华民族伟大复兴的必然要求。
  </span>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <br/>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <strong>
   <span style="font-size: 16px;color: rgb(0, 122, 170);">
    第一，携手推动民族复兴，实现和平统一目标。
   </span>
  </strong>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <br/>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <span style="color: rgb(86, 86, 86);font-size: 16px;">
   一水之隔，咫尺天涯，两岸迄今尚未完全统一是历史遗留给中华民族的创伤。两岸中国人应该共同努力谋求国家统一，抚平历史创伤。
  </span>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <br/>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <span style="color: rgb(86, 86, 86);font-size: 16px;">
   在中华民族走向伟大复兴的进程中，台湾同胞定然不会缺席。
  </span>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <br/>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <span style="color: rgb(86, 86, 86);font-size: 16px;">
   台湾前途在于国家统一，台湾同胞福祉系于民族复兴。台湾问题因民族弱乱而产生，必将随着民族复兴而终结！
  </span>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <br/>
 </p>

 </p>

 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <strong>
   <span style="font-size: 16px;color: rgb(0, 122, 170);">
    第二，探索“两制”台湾方案，丰富和平统一实践。
   </span>
  </strong>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <br/>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <span style="color: rgb(86, 86, 86);font-size: 16px;">
   “和平统一、一国两制”是实现国家统一的最佳方式。
  </span>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <br/>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <span style="color: rgb(86, 86, 86);font-size: 16px;">
   制度不同，不是统一的障碍，更不是分裂的借口。
  </span>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <br/>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <span style="color: rgb(86, 86, 86);font-size: 16px;">
   “一国两制”在台湾的具体实现形式会充分考虑台湾现实情况，会充分吸收两岸各界的意见和建议，会充分照顾到台湾同胞利益和感情。
  </span>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <br/>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <span style="color: rgb(86, 86, 86);font-size: 16px;">
   两岸同胞是一家人，两岸的事是两岸同胞的家里事，当然也应该由家里人商量着办。
  </span>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <br/>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <span style="color: rgb(86, 86, 86);font-size: 16px;">
   和平统一，是平等协商、共议统一。两岸长期存在的政治分歧问题是影响两岸关系行稳致远的总根子，总不能一代一代传下去。
   <br/>
  </span>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <br/>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <span style="color: rgb(86, 86, 86);font-size: 16px;">
   以对话取代对抗、以合作取代争斗、以双赢取代零和，两岸关系才能行稳致远。
  </span>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <br/>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <span style="color: rgb(86, 86, 86);font-size: 16px;">
   我们郑重倡议，在坚持“九二共识”、反对“台独”的共同政治基础上，两岸各政党、各界别推举代表性人士，就两岸关系和民族未来开展广泛深入的民主协商，就推动两岸关系和平发展达成制度性安排。
  </span>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <br/>

  <strong>
   <span style="font-size: 16px;color: rgb(0, 122, 170);">
    第三，坚持一个中国原则，维护和平统一前景。
   </span>
  </strong>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <br/>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <span style="color: rgb(86, 86, 86);font-size: 16px;">
   统一是历史大势，是正道。“台独”是历史逆流，是绝路。
  </span>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <br/>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
  <span style="color: rgb(86, 86, 86);font-size: 16px;">
   我们愿意为和平统一创造广阔空间，但绝不为各种形式的“台独”分裂活动留下任何空间。
  </span>
 </p>
 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">

 <p style="margin-right: 8px;margin-left: 8px;white-space: normal;line-height: 1.75em;text-indent: 2em;">
