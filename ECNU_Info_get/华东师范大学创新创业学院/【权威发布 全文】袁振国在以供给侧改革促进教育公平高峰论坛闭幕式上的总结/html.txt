
 <p style="line-height: 25.6px; white-space: normal;">
  <span style="line-height: 25.6px; color: rgb(122, 68, 66); font-size: 12px;">
   袁振国
  </span>
 </p>
 <p style="line-height: 25.6px; white-space: normal;">
  <span style="color: rgb(122, 68, 66); font-size: 12px;">
   华东师范大学终身教授、中国教育学会副会长
  </span>
 </p>
 <p style="line-height: 25.6px; white-space: normal;">
  <br/>
 </p>
 <p style="white-space: normal; line-height: 24px; text-align: justify; text-indent: 2em;">
  <br/>
 </p>
 <p style="text-indent: 32px; line-height: 150%; text-align: justify;">
  <span style="font-size:16px;line-height:150%;font-family:微软雅黑,sans-serif">
   今天有500多位来自全国17个省市自治区的学者、同仁参加会议，有20位专家分6个议题做了专题的发言。专家们的观点精彩纷呈，面广量大，也有不少交锋，所以，要对会议进行总结是一件困难的事情，我只能从我个人的角度谈一点感受和想法。
  </span>
 </p>
 <p style="text-indent: 32px; line-height: 150%; text-align: justify;">
  <span style="font-size:16px;line-height:150%;font-family:微软雅黑,sans-serif">
   “以供给侧改革来推动教育公平”可以从不同角度进行探讨，这次会议大家重点在这四个方面或四个层次展开了讨论，对教育公平的认识也在很多方面有所深入，形成了一系列重要成果。第一，从供给侧改革看教育公平大大拓展了促进教育公平的思路；第二，拓展了从经济社会发展整体结构中考虑教育改革发展以促进教育公平的宏观政策思考；第三，深入探讨了教育自身的结构性变革对促进教育公平的意义；第四，讨论了教育公平从宏观层面向微观层面的深化。
  </span>
 </p>
 <p style="text-indent: 39px; line-height: 150%; text-align: justify;">
  <strong>
   <span style="font-size:16px;line-height:150%;font-family:微软雅黑,sans-serif">
    一、供给侧改革的根本目的是改善供求关系，是解决怎么样才能更好地满足人的发展需要和社会发展的需要。
   </span>
  </strong>
  <span style="font-size:16px;line-height:150%;font-family:微软雅黑,sans-serif">
   进行供给侧的改革，从促进教育公平的角度来说，核心是怎么样满足更多、更好、更合理的教育需要。
  </span>
 </p>
 <p style="margin-left: 31px; line-height: 150%; text-align: justify;">
  <span style="font-size:16px;line-height:150%;font-family:微软雅黑,sans-serif">
   首先是教育的基本需要。教育是全民的事业，教育首先要满足的是最基本
  </span>
 </p>
 <p style="line-height: 150%; text-align: justify;">
  <span style="font-size:16px;line-height:150%;font-family:微软雅黑,sans-serif">
   的、最广大民众最基础的需要。这个需要是什么？对任何一个国家来说，首先是强调和保障义务教育。我国现在实行的是9年制义务教育，那么，9年义务教育能不能保证面向所有儿童，有没有同样的标准，能不能保证同等的质量？显然还有这样那样的问题和差距，因此《国家中长期教育改革和发展规划纲要（2010-2020）》把推进义务教育的均衡发展作为首要的和战略的任务。这次我们很多同仁讨论了儿童健康、儿童营养、儿童发育，这都是义务教育里面的应有之意。要保证在同一个蓝天下享受同等的教育。进城务工人员的孩子，还有相当一部分没有受到完整的义务教育，有一些虽然受完了九年义务教育，但是享受的教育资源完全不可同日而语。还有这次涉及的比较少但非常重要的，是对残疾儿童、特殊困难儿童的教育。对于有特殊需要的孩子来说，不仅要享受同等水平的教育，而且要给他们以特殊的帮助。如果只给他们机会没有特殊的帮助，依然不能正常发展。所以，对于经济困难的孩子，对于特殊困难的孩子，怎么样从政策上给予必要的保障，这也是要深入研究的。这几年，推出了很多措施来提高底部，今天有学者特别强调了精准扶贫，都是强调的满足这种需要。
  </span>
 </p>
 <p style="text-indent: 32px; line-height: 150%; text-align: justify;">
  <span style="font-size:16px;line-height:150%;font-family:微软雅黑,sans-serif">
   第二是不同类型的需要。一个国家并不需要把所有人都培养成科学家或博士，这句话的含义是什么呢？就是社会发展的过程当中需要各种各样的人，需要通过不同类型的教育完成不同的培养任务。但类型不是等级，如普通教育和职业教育，基础教育和高等教育，工科、理科、文科、商科不同专业教育。我们教育有很多类型，而这些类型都是社会发展所需要的，也是每个人根据自己的特点可以做选择的。但是现在我们在不同类型的教育问题上有一个普遍性的问题，就是把类型教育搞成了等级教育，本来应该受到同样支持、受到同样尊严的教育，变成了三六九等，这自然就拉大了差距。这是这次讨论当中谈到比较多的。
  </span>
 </p>
 <p style="text-indent: 32px; line-height: 150%; text-align: justify;">
  <span style="font-size:16px;line-height:150%;font-family:微软雅黑,sans-serif">
   第三个就是要满足有选择的个性化的教育。教育的发展和人的发展需要因材施教。从宏观的层面到微观的教学层面大家谈得很深入，而且是有大量数据支撑的研究。那些有杰出才华、有创新能力的，包括偏才、怪才，需要特殊的环境和机制。一统化和标准化只适用高考分数，而且分数本身也有很多弊端，分数和分数也不一样，这些问题使得因材施教很难实现，这对国家的发展也不利。
  </span>
 </p>
 <p style="text-indent: 32px; line-height: 150%; text-align: justify;">
  <span style="font-size:16px;line-height:150%;font-family:微软雅黑,sans-serif">
   第四，要满足教育发展的新需要。特别是在网络化背景下，很多教育新形态、新样式、新模式纷至沓来。在这种背景下，我们该怎么利用这种形势，怎样利用这种资源，避免弊端，充分发挥有利的优势和条件，这次讨论给了我们很深刻的印象。比如信息化可能促进教育公平，也有可能扩大教育差距。
  </span>
 </p>
 <p style="text-indent: 32px; line-height: 150%; text-align: justify;">
  <span style="font-size:16px;line-height:150%;font-family:微软雅黑,sans-serif">
   供给侧改革这个概念是从经济学上延伸过来的。教育是一项公共性事业，和产品的销售和商业的发展有相同的地方，但是也有不同的地方。作为一项公共性事业，教育的需要和经济、商业的发展有一个最大的不同，就是不仅要满足个人的消费需要，而且要考虑社会发展的需要。而社会发展的需要和个人发展的需要之间总体上来说，从长远的角度来说是一致的，但是在不同的特定历史阶段和具体内容上有各种各样的矛盾和冲突。我们今天讨论教育公平的时候还没有太注意这个问题。如果对这个问题不深刻认识的话，对教育公平的认识是不全面的，需要把这两者紧密结合起来，完善和制定促进教育公平的国家及地方政策。
  </span>
 </p>
 <p style="text-indent: 32px; line-height: 150%; text-align: justify;">
  <strong>
   <span style="font-size:16px;line-height:150%;font-family:微软雅黑,sans-serif">
    第二，从供给侧改革的宏观角度讨论教育公平，需要调动和整合全社会的资源发展教育，跳出教育看教育。
   </span>
  </strong>
  <span style="font-size:16px;line-height:150%;font-family:微软雅黑,sans-serif">
   尽管通过市场的调节促进教育的体制改革和创新是主要的力量，但教育公平这件事情是不能交给市场的。要公平找政府，要选择找市场。如果我们把公平的问题交给市场处理，那就无公平可言了。在供给侧改革这样一个大的战略思想背景下，如何加深对满足教育的社会需要和个人需要的认识和能力，这次形成了很多重要共识。
  </span>
 </p>
 <p style="text-indent: 32px; line-height: 150%; text-align: justify;">
  <span style="font-size:16px;line-height:150%;font-family:微软雅黑,sans-serif">
   很多教育的问题绝不仅仅是教育自身的问题。今天会议上谈到经费的问题、土地的问题、编制的问题、指标的问题等等，地方的财政怎么分配，地方土地怎么使用，这些制约教育的重要因素，都不是教育部门可以解决的。现在我国的教育经费财政已经达到是2.67万亿，这是一个相当大规模的数字，但是80%以上是地方财政。东、中、西部的差距为什么这么大，因为发展不平衡，没有中央转移支付，很多地方教育就很难运转，比如说甘肃的财政收入，只占财政支出的50%不到，换句话说，一半以上的支出要靠转移支付。转移支付在地方之间分配怎么样更加合理，经费在地方上怎么使用，这就不是教育内部能解决得了的。现在城镇化继续深入，特别是北京、上海、深圳、广州这样的特大型城市，采取了积分制，设了一个门槛，不让外地人进来，现在是挡住了，十年以后，二十年以后呢？老龄化的程度越来越高，2030年人口结构是什么样？人口一旦老龄化要调整是很难的，怎么样从长远的角度看问题，让城市有活力？国际上的大城市没有一个是用这种办法管理的。这不是教育部能够解决的了的。这些问题一定要全社会来关注，要靠调整城市发展战略，靠整个社会进行综合改革。
  </span>
 </p>
 <p style="text-indent: 32px; line-height: 150%; text-align: justify;">
  <strong style="line-height: 150%; text-indent: 31px;">
   <span style="line-height: 150%; font-family: 微软雅黑, sans-serif;">
    第三，说到教育内部的供给问题，首先是结构性调整的问题。
   </span>
  </strong>
  <span style="line-height: 150%; text-indent: 31px; font-family: 微软雅黑, sans-serif;">
   教育有各种各样的结构，类型结构、层次结构、学科结构、专业结构等等，这里我想特别强调类型结构。类型结构当中有普通教育和职业教育，公办教育和民办教育，学校教育和社会教育。普通教育和职业教育之间，职业教育支持不够。我们大家充分认识到职业教育发展好很必要，但是有几个人愿意上职业学校呢？我们在座的各位嘉宾愿意把你们的孩子送到职业学校去吗？如果我孩子实在考不好，只好去，但是内心是不愿意的。为什么不愿意呢？因为普通教育和职业教育的待遇不同。职业学校发展经费是不足的，师资质量、办学标准都是不一样的，这个结构要慢慢调整，让职业教育成为受人尊敬、成为志愿选择的教育，而不是说你不行你就要去职业学校。第二是公办教育和民办教育的问题。现在确实存在一个国进民退的问题，我们大学在经费比较缺乏的时候国家财政只占40%，各种各样的经费来源多的时候也只占60%，现在一般的高等学校财政支出都在80%以上。这表明了我们政府的财政雄厚的实力和支持高等教育发展的决心。从另外一个角度来说，我们的民间资本在教育上越来越少，为什么？社会总财富获得了增加，国民产值已经达到了68万亿，这在三十年前是不能想象的事情。现在百亿富翁已有不少，甚至千亿富翁也有，他们为什么不愿意把钱放在教育上，肯定是政策上出了问题。我们对民办教育到底怎么看？怎么样让更多的人捐资也好，出资也好，合作方式办学也好，我觉得无论是对国家还是对个人来说都是有好处的。第三个是学校教育和社会教育，这一点我们今天谈得少，基本上没有谈。实际上要讲起这个问题来，比前面两个问题更严重。我们基本上讲的是学校教育的问题。其实从OECD的报告来看，一个国家的核心竞争力是什么呢？OECD研究表明，一个国家在现代社会的核心竞争力是在职人员的劳动技能。劳动技能能不能发展，在职人员能不能掌握新的劳动技能，这是一个国家的核心竞争力。因此，继续教育非常重要。继续教育的经费在我们整个教育盘子里面无论是中央还是地方占的比例都非常少。说到公平问题根本没法讨论，太多的人根本得不到进修的机会。太多的人得不到必要的再教育机会。
  </span>
 </p>
 <p style="text-indent: 31px; line-height: 150%; text-align: justify;">
  <strong style="line-height: 150%;">
   <span style="line-height: 150%; font-family: 微软雅黑, sans-serif;">
    第四，对隐性的教育公平问题做了深入的讨论。
   </span>
  </strong>
  <span style="line-height: 150%; font-family: 微软雅黑, sans-serif;">
   在师生关系上、课程、作业、座位排次这方面的研究是空前的。这反映了我们一个重要的概念，即有质量的公平。我们不是说一般意义的公平，而是到了有质量的公平的新阶段。对一个学生来讲，对某一个具体的人来说，在课堂上在学校里得到的公平待遇，比起在哪个学校上学更重要。从宏观上来说，学校和学校之间的差别，地区和地区之间的差别，城市和乡村的城乡差距要通过宏观来调整，但是对于一个具体的孩子来说，面对的是在这个特定的环境当中生活九年甚至十二年，这个过程对他来说其实比宏观上的差距更现实、更直接，影响也更深远。这次特别强调了教学过程当中在微观层次上的公平问题，一定会对我们进一步深化教育公平问题起到积极的推动作用。
  </span>
 </p>
 <p style="text-indent: 31px; line-height: 150%; text-align: justify;">
  <span style="font-family: 微软雅黑, sans-serif; line-height: 150%;">
   这是我的粗浅的认识。最后，我代表会议的主办单位向与会代表的支持特别是嘉宾的精彩发言表示衷心的感谢，向付出了辛苦劳动的会务人员和几十位志愿者表示衷心的感谢！

 </p>
