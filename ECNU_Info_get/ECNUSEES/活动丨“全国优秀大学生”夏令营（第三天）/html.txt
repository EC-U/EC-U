     <p style="text-indent: 24px;">
      <span style="color: rgb(0, 0, 0); font-size: 14px;">
       <span style="color: rgb(0, 0, 0);">
        7
       </span>
       <span style="color: rgb(0, 0, 0); font-family: 宋体;">
        月
       </span>
       <span style="color: rgb(0, 0, 0);">
        14
       </span>
       <span style="color: rgb(0, 0, 0); font-family: 宋体;">
        日，是本次夏令活动的第三天，也是营员们来到长江口湿地生态系统野外监测研究站（崇西分站）的第二天。原计划上午组队进行的西沙湿地野外生态考察，因为一场瓢泼大雨被迫取消。为了不让营员们失望，刘文亮老师临时决定在会议厅开展西沙湿地生态讲座，让同学们不出门也能了解到关于西沙湿地的生态知识。讲座分为底栖动物和植物两大部分，刘老师幽默健谈的授课风格使整场讲座格外生动有趣，营员们在欢声笑语中掌握到了很多知识。

      </span>

     <p style="text-indent: 24px;">
      <span style="font-family: 宋体; color: rgb(0, 0, 0); font-size: 14px;">
       午饭过后，营员们早早地来到了会议厅，根据之前在标本馆的学习经历，整理资料为下午的汇报任务进行准备。根据营员的专业背景及兴趣爱好，汇报小组分为植物组、底栖动物组、和鸟类组。各小组团结一致又分工明确，组织起了热切的讨论。
      </span>
     </p>
     <p style="text-indent: 24px;">
      <span style="color: rgb(0, 0, 0); font-size: 14px;">

       </span>
       <span style="color: rgb(0, 0, 0);">
        3
       </span>
       <span style="color: rgb(0, 0, 0); font-family: 宋体;">
        点，汇报正式开始。首先，鸟类组通过一个生动的童话故事将长江湿地的几种常见鸟的长相，习性以及种间关系形象地展现出来；接着，植物组以严谨的科研态度，扎实的学术背景给大家介绍了当地植物的特征用途等；最后，底栖动物组也对本组工作成果依次进行了展示。各位营员凭借生动的语言、清晰的思路，赢得了老师和同学们的阵阵掌声。
