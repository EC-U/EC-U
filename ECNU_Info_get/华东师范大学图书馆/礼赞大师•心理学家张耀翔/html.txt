      <p style="margin: 0px;padding: 0px;box-sizing: border-box;">
       <strong style="box-sizing: border-box;">
        中国提倡心理学第一人
       </strong>

   <section style="box-sizing: border-box;">
    <section style="line-height: 2;font-size: 14px;box-sizing: border-box;">
     <p style="text-indent: 2em;margin: 0px;padding: 0px;box-sizing: border-box;">
      张耀翔（1893.2.23-1964.7.9），心理学家。湖北汉口（今武汉）人。1915年清华学校留美预备部毕业赴美国，先后就读于安麦斯学院（Amherst College）和哥伦比亚大学，1919年获硕士学位（心理学）。1920年应北京高等师范学校校长陈宝泉邀请回国，任北京高等师范学校、北京师范大学教授、教育研究科主任。1928年任大夏大学教育科主任，从此移居上海，历任大夏、暨南、南京中央大学、光华、复旦等校教育学/心理学教授、系主任。
      <strong style="box-sizing: border-box;">
       1951年加入新成立的华东师范大学，
      </strong>
      直至1964年因病去世
      <strong style="box-sizing: border-box;">
       。
      </strong>
     </p>
    </section>
   </section>
  </section>

    <section style="font-size: 14px;box-sizing: border-box;">
     <p style="text-indent: 2em;margin: 0px;padding: 0px;box-sizing: border-box;">
      <span style="color: rgb(23, 24, 22);box-sizing: border-box;">
       <strong style="box-sizing: border-box;">
        张耀翔先生1921年
       </strong>
      </span>
      <strong style="box-sizing: border-box;">
       <span style="color: rgb(23, 24, 22);box-sizing: border-box;">
        参与组建中国第一个心理学组织中华心理学会并被推举为会长，1922年创办中国乃至东方第一种心理学杂志《心理》并担任主编。
       </span>
      </strong>
      先生运用心理学原理观察、分析不同阶层、不同类型的人所产生的心理活动，对人类的生理、性格、情绪、智慧、才能形成的特点、遗传以及社会犯罪等心理活动过程，都有探讨和论述。先生从1930年代开始搜集中国心理学史资料，若断若续达十余年，惜未能成书，只在1940年发表了文章
      <strong style="box-sizing: border-box;">
       《中国心理学的发展史略》
      </strong>
      ，这也是其晚年遗憾之一。
     </p>
    </section>

     <p style="text-indent: 2em;margin: 0px;padding: 0px;box-sizing: border-box;">
      张耀翔先生热心社会活动，
      <strong style="box-sizing: border-box;">
       曾任华东师范大学工会副主席/主席达十年之久
      </strong>
      。民国年间，先生除在多所高校任职、兼课外，还参与各种社会组织及其活动，如非宗教大同盟、中华教育改进社、中华职业教育社、中华学艺社、平民教育促进会、上海图书馆协会、上海心理卫生委员会、上海职业指导所、中国文化建设协会等。其活动在报刊上有大量提及，以下云图可窥其个人社会网络之一斑。

    <section style="font-size: 14px;box-sizing: border-box;">
     <p style="text-indent: 2em;margin: 0px;padding: 0px;box-sizing: border-box;">
      演讲是张耀翔先生社会活动的重要内容，他的不少报刊文章，本身就是演讲记录。1921年11月3日《申报》报道“京高师平民教育社讲演会”，描述了先生演讲“考试新法”时的盛况与演讲方式：到会听讲之人特多、该校雨中操场几不能容首；演讲长达2小时，在没有PPT的时代，先生采用“用纸列表表示于板上”的方法，向听众展示需要说明的材料。

      <p style="margin: 0px;padding: 0px;box-sizing: border-box;">
       <strong style="box-sizing: border-box;">
        “人文计算/数字人文”的先行者
       </strong>

     <section style="max-width: 100%;display: inline-block;vertical-align: top;line-height: 0;width: 1em;box-sizing: border-box;">
      <img class="raw-image" data-ratio="1" data-type="png" data-w="100" src="https://mmbiz.qpic.cn/mmbiz_png/hXkAw6fUrNXiak7ZLCwoiaibX7MSwUIrXm6VNS0qYkJF0nBEEOhBSSXyN3hficaVDUWNJMQ3dcqRPpsoJBhf4yvmRQ/640?wx_fmt=png" style="vertical-align: middle;max-width: 100%;width: 100%;box-sizing: border-box;" width="100%"/>

   <section style="box-sizing: border-box;">
    <section style="box-sizing: border-box;">
     <p style="text-indent: 2em;margin: 0px;padding: 0px;box-sizing: border-box;">
      <span style="font-size: 14px;box-sizing: border-box;">
       心理学研究之外，
       <strong style="box-sizing: border-box;">
        先生还采用统计方法研究其他学科，堪称“人文计算/数字人文”的先行者
       </strong>
       。由于视角独特，往往引起争议。特别受到关注的有：
      </span>
     </p>
     <p style="text-indent: 2em;margin: 0px;padding: 0px;box-sizing: border-box;">
      <strong style="box-sizing: border-box;">
       <span style="font-size: 14px;box-sizing: border-box;">
        清代进士之地理的分布
       </span>
      </strong>
      <span style="font-size: 14px;box-sizing: border-box;">
       （演讲题为：中国人才产生地；1926年）。此文列名中国历史统计学的重要著作。
      </span>
     </p>
     <p style="text-indent: 2em;margin: 0px;padding: 0px;box-sizing: border-box;">
      <strong style="box-sizing: border-box;">
       <span style="font-size: 14px;box-sizing: border-box;">
        新诗人之情绪
       </span>
      </strong>
      <span style="font-size: 14px;box-sizing: border-box;">
       （1924年）
      </span>
      <span style="font-size: 14px;box-sizing: border-box;">
       。统计当时最著名的10部白话诗集中的感叹符（包括胡适《尝试集》、郭沫若《女神》、冰心《繁星》等），结果是平均每四行诗一个“！”，与公认之外国好诗相比多六倍。结论中引子夏毛诗序之后，称“若今之白话诗，可谓亡国之音矣”。由此引起争议，以至鲁迅也在多篇杂文中间接予以嘲讽。
      </span>
     </p>
     <p style="text-indent: 2em;margin: 0px;padding: 0px;box-sizing: border-box;">
      <strong style="box-sizing: border-box;">
       <span style="font-size: 14px;box-sizing: border-box;">
        六百年来最有势力的小学校教科书
       </span>
      </strong>
      <span style="font-size: 14px;box-sizing: border-box;">
       （1925年
      </span>
      <strong style="box-sizing: border-box;">
       <span style="font-size: 14px;box-sizing: border-box;">
        ）
       </span>
      </strong>
      <span style="font-size: 14px;box-sizing: border-box;">
       。本文通过比较，表明《三字经》比民国年间新编《平民千字课》收录用字更合理；又统计《三字经》内容涉及历史、教育等十五种学科，只有体育、卫生没有涉及。结论说明“这部书虽不是理想的教科书，但是有的地方确比新出的教科书强得多”。对《三字经》内容的称赞又引发一波争议。
      </span>
     </p>
    </section>
