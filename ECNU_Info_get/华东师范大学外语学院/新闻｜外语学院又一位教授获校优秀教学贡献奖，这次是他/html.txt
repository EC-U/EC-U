
 <p style="line-height: 150%;text-indent: 0em;">
  <span style="line-height: 150%;font-family: 微软雅黑, sans-serif;color: rgb(0, 0, 0);font-size: 16px;">
   2019年1月3日下午，华东师范大学召开本科教学总结表彰大会，在新年伊始见证与分享本科教学的荣耀和喜悦。外语学院德语系系主任宋健飞教授荣获2018年优秀教学贡献奖。自2014年学校设立教学贡献奖以来，每年都有一位教授获得该本科教学最高荣誉。另外，英语系全建强副教授荣获2018年度本科教学年度贡献奖。

  <span style="line-height: 22.5px;font-family: 微软雅黑, sans-serif;color: rgb(0, 0, 0);font-size: 15px;">
  </span>
  <span style="line-height: 22.5px;font-family: 微软雅黑, sans-serif;color: rgb(0, 0, 0);font-size: 16px;">
   华东师大优秀教学贡献奖获得者宋健飞教授在采访中说道：“作为一名从事中德翻译教学的老师，
   <strong>
    我心目中的理想课堂
   </strong>
   是能够建构一个中德教师同台合作授课的教学环境，充分发挥双方母语文化背景的优势，分工合作，形成真实的跨文化交际语境，让学生切身感受和体验中德两种语言在碰撞中所呈现出的各种可能，深刻理解两者相互转换过程中的异同、难点、障碍以及种种意料之中和意料之外的因素，学会在实践中解决问题的方法。
  </span>
 </p>
 <p style="text-indent: 0px;white-space: normal;line-height: 25.5px;">
  <span style="color: rgb(0, 0, 0);font-family: 微软雅黑, sans-serif;font-size: 16px;text-indent: 28px;">
   德国著名汉学家、翻译家、中华图书特殊贡献奖获得者高立希（Ulrich Kautz）先生是我景仰的良师益友。他将自己丰富的实践经验用于教学，理论联系实际，学贯东西，培养了许多中德教师和翻译工作者，其治学严谨，精益求精，深入浅出，循循善诱，诲人不倦之风范，至今仍深深地影响着我。”
  </span>
