 <p style="margin-top:10px;margin-right:0;margin-bottom:0;margin-left:0;text-indent:28px;padding:0 0 0 0;line-height:21px;">
  <span style="font-size: 15px;letter-spacing: 1px;">
   <strong>
    <span style="font-size: 15px;font-family: 宋体;">
     讲座介绍：
    </span>
   </strong>
   <span style="font-size: 15px;font-family: 宋体;">
    <span style="font-size: 15px;font-family: 宋体;">
     中国电影发展史和世界电影史一样，最初出现的是纪实影片。
    </span>
    1949年以前，有关中国题材的记录影片，绝大多数是外国机构或外国人拍摄的。
   </span>
  </span>
 </p>
 <p style="margin-top:10px;margin-right:0;margin-bottom:0;margin-left:0;text-indent:28px;padding:0 0 0 0;line-height:21px;">
  <span style="font-family: 宋体;font-size: 15px;letter-spacing: 1px;">
   外国机构或外国人拍摄的中国与中国人，自然与我们中国人自己拍摄的视角不同：当年我们的前人司空见惯、见怪不怪的景象，在洋人的镜头里就成了稀奇古怪、新鲜猎奇的画面……
  </span>
 </p>
 <p style="margin-top:10px;margin-right:0;margin-bottom:0;margin-left:0;text-indent:28px;padding:0 0 0 0;line-height:21px;">
  <span style="font-family: 宋体;font-size: 15px;letter-spacing: 1px;">
   恰恰是那些外国机构或外国人当年以猎奇的眼光记录下来的遗留在胶片上的影像，成就了我们当下认识、解读当年中国社会和中国人的重要史料与档案。
  </span>
 </p>
 <p style="margin-top:10px;margin-right:0;margin-bottom:0;margin-left:0;text-indent:28px;padding:0 0 0 0;line-height:21px;">
  <span style="font-family: 宋体;font-size: 15px;letter-spacing: 1px;">
   然而，电影诞生的一百多年来，在中国乃至世界范围内，由于战争、天灾和社会动荡诸因素，能够完好留存下来的早期电影胶片数量并不太多，尤其是由个人或小型且存续短暂之机构拍摄的独特而稀少的电影胶片更是凤毛麟角。
  </span>
 </p>
 <p style="margin-top:10px;margin-right:0;margin-bottom:0;margin-left:0;text-indent:28px;padding:0 0 0 0;line-height:21px;">
  <span style="font-family: 宋体;font-size: 15px;letter-spacing: 1px;">
   2012年以来，主讲人所在的机构——上海音像资料馆利用收藏市场与文物拍卖平台，发现了一大批遗留在民间和海外的老记录电影胶片，通过有偿征集与采购，并经过整理、修复、数字化和初步的研究，获得了大量的早期中国题材的珍贵历史影像。
  </span>
 </p>
 <p style="text-indent: 2em;">
  <span style="font-family: 宋体;font-size: 15px;letter-spacing: 1px;">
   12月17日晚上17:00，主讲人龚伟强老师将带着三部珍贵的影像视频来与同学们分享。
  </span>
 </p>
