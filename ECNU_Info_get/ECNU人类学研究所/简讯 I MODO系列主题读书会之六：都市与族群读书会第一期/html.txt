   </span>
   “都市与族群读书会”的两个关键词，“都市”和“族群”做了简要分享。
  </span>
 </p>
 <p style="text-indent: 32px;text-align: justify;line-height: 1.75em;">
  <span style="font-family:宋体;font-size:16px;">
   <span style="font-family:宋体;">
    随后，来自社会学系的硕士研究生张荣瑾同学为大家简要梳理了社会学领域有关城市研究的主要脉络，并与其他几位同学分别就《城市
   </span>
   ——
   <span style="font-family:宋体;">
    有关城市环境中人类行为研究的建议》（
   </span>
   <span style="font-family:Times New Roman;">
    2006
   </span>
   <span style="font-family:宋体;">
    ）、《城市意象》（
   </span>
   <span style="font-family:Times New Roman;">
    2011
   </span>
   <span style="font-family:宋体;">
    ）、《苏北人在上海：
   </span>
   <span style="font-family:Times New Roman;">
    1850-1980
   </span>
   <span style="font-family:宋体;">
    》（
   </span>
   <span style="font-family:Times New Roman;">
    2004
   </span>
   <span style="font-family:宋体;">

   </span>
   <span style="font-family:Times New Roman;">
    20
   </span>

   </span>
   <span style="font-family:Times New Roman;">

   </span>
   <span style="font-family:宋体;">

   </span>
  </span>
 </p>
 <p style="text-align: justify;line-height: 1.75em;margin-bottom: 10px;margin-top: 10px;text-indent: 2em;">
  <span style="font-family: 宋体;font-size: 16px;text-align: left;text-indent: 32px;">
   临近读书会尾声，刘琪副教授向同学们讲述了推荐上述书目作为
  </span>
  <span style="font-family: 宋体;font-size: 16px;text-align: left;text-indent: 32px;">
   “都市与族群读书会第一期”的初衷。刘琪副教授认为，人类学研究强调文化或人文，这既是我们的学科长项，但有时也难免会造成我们观察事物的盲点。因为，一个城市不仅有形色各异的人，也耸立着星罗棋布的建筑物，他们相互作用，共同构成了城市。空间、景观或物也理当被纳入观察的范围。因此扩大阅读范围，吸纳多学科知识，扩宽研究视野势必会使我们的研究更加丰腴。
  </span>
 </p>
