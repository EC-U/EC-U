  <p style="text-indent: 2em;">
   <span style="font-size: 15px;color: rgb(127, 127, 127);">
    美国弗吉尼亚大学作为华东师范大学战略合作伙伴，多年来与师大保持多维度、多层面的良好交流。本次来访的Hereford书院是
   </span>
   <span style="font-size: 15px;color: rgb(2, 30, 170);">
    <strong>
     弗吉尼亚三大住宿制书院之一
    </strong>
   </span>
   <span style="font-size: 15px;color: rgb(127, 127, 127);">
    ，秉持弗吉尼亚大学
   </span>
   <span style="font-size: 15px;color: rgb(2, 30, 170);">
    <strong>
     “Learn to live, live to learn”
    </strong>
   </span>
   <span style="font-size: 15px;color: rgb(127, 127, 127);">
    的住宿制书院办理理念，以
   </span>
   <span style="font-size: 15px;color: rgb(255, 169, 0);">
    <strong>
     丰富的社区生活、多样的项目体验、交融的文化环境
    </strong>
   </span>
   <span style="font-size: 15px;color: rgb(127, 127, 127);">
    为特色。
   </span>
  </p>
  <p style="text-indent: 2em;">
   <span style="font-size: 15px;color: rgb(127, 127, 127);">
    去年3月，孟院首次接待Hereford书院，短短1天的活动安排给UVA到访的师生留下了深刻印象，双方达成了长期合作的初步意向，并于去年下半年签署了项目合作协议。今年，Hereford书院10名师生将在中国开展
   </span>
   <span style="font-size: 15px;color: rgb(2, 30, 170);">
    <strong>
     为期6天的文化考察
    </strong>
   </span>
   <span style="font-size: 15px;color: rgb(127, 127, 127);">
    ，孟宪承书院全程参与了活动的设计与安排。本次
   </span>
   <span style="font-size: 15px;color: rgb(2, 30, 170);">
    <strong>
     两家书院学生共同合作参与的主题体验式学习
    </strong>
   </span>
   <span style="font-size: 15px;color: rgb(127, 127, 127);">
    将作为Hereford书院今年中国之行的重点行程。
   </span>
