    </span>
    <br/>
   </p>
   <p>
    <span style="color: rgb(61, 170, 214);">
     <strong style='font-size: 15px;font-family: -apple-system-font, BlinkMacSystemFont, "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;'>

  <p style="text-align: justify;">
   <span style="font-size: 15px;color: rgb(61, 170, 214);">
    <strong>
     <strong style="font-size: 15px;white-space: normal;">
      <span style="font-size: 15px;color: rgb(61, 170, 214);font-family: 宋体;">
      </span>
     </strong>
    </strong>
   </span>
  </p>
  <p style="margin: 0cm 8px 0.0001pt;line-height: 20.4pt;background: white;">
   <span style="color: rgb(0, 122, 170);font-size: 15px;">
    <strong>
     <span style="color: rgb(0, 122, 170);font-family: 宋体;letter-spacing: 0.4pt;">
      ◆ 翟天临事件后，这门新课的推出是“蹭热点”吗？
     </span>
    </strong>
   </span>
  </p>
  <p style="margin: 0cm 8px 0.0001pt;line-height: 20.4pt;background: white;">
   <span style="font-size: 15px;">
    <strong>
     <span style="font-size: 15px;font-family: 宋体;letter-spacing: 0.4pt;">
      <br/>
     </span>
    </strong>
   </span>
  </p>
  <p style="margin: 0cm 8px 0.0001pt;line-height: 20.4pt;background: white;">
   <span style="font-size: 15px;">
    <strong>
     <span style="font-size: 15px;font-family: 宋体;letter-spacing: 0.4pt;">
      萧
     </span>
    </strong>
    <span style="font-size: 15px;letter-spacing: 0.4pt;">
     ：当下学术规范和伦理问题越来越突出，这门课程的开设契合了社会及学校的需求，但开设这门课并非“赶时髦”“蹭热点”之举，该课程设计策划已有一年多时间了，做了诸多准备工作。这门课是由校研究生院提出设立的，又组织了专家评委会论证，专家们当场就提出了许多好的建设性意见。
    </span>
   </span>
  </p>
  <p style="margin: 0cm 8px 0.0001pt;line-height: 20.4pt;background: white;">
   <span style="letter-spacing: 0.4pt;font-size: 15px;">
    <br/>
   </span>
  </p>
  <p style="margin: 0cm 8px 0.0001pt;white-space: normal;line-height: 20.4pt;background: white;">
   <span style="color: rgb(0, 122, 170);font-size: 15px;">
    <strong>
     <span style="color: rgb(0, 122, 170);font-size: 15px;font-family: 宋体;letter-spacing: 0.4pt;">
      ◆ 您怎么看待学术不端现象？
     </span>
    </strong>
   </span>
  </p>
  <p>
   <br/>
  </p>
  <p style="margin: 0cm 8px 0.0001pt;line-height: 20.4pt;background: white;">
   <span style="letter-spacing: 0.4pt;text-indent: 24.6pt;font-size: 15px;">
    <strong style="font-size: 15px;white-space: normal;background-color: rgb(255, 255, 255);">
     <span style="font-family: 宋体;letter-spacing: 0.4pt;">
      萧
     </span>
    </strong>
    <span style="font-size: 15px;background-color: rgb(255, 255, 255);letter-spacing: 0.4pt;">
     ：
    </span>
    从历史看，最早的学术交流其实就是某一专业的知识人士就大家共同感兴趣的学术议题所做的“沙龙”讨论，所以报告人都要展示自己最前沿的学术思想和学术设想。
   </span>
  </p>
  <p style="margin: 0cm 8px 0.0001pt;line-height: 20.4pt;background: white;">
   <span style="letter-spacing: 0.4pt;text-indent: 24.6pt;font-size: 15px;">
    <br/>
   </span>
  </p>
  <p style="margin: 0cm 8px 0.0001pt;line-height: 20.4pt;background: white;">
   <span style="letter-spacing: 0.4pt;text-indent: 24.6pt;font-size: 15px;">
    可以想象，在这种性质的环境中根本就不可能出现什么
“造假”和“抄袭”问题。但随着现代印刷技术的出现，特别是计算机的迅速发展，我们进入到一个符号“仿真”和“复制”的时代。
   </span>
  </p>
  <p style="margin: 0cm 8px 0.0001pt;line-height: 20.4pt;background: white;">
   <span style="letter-spacing: 0.4pt;text-indent: 24.6pt;font-size: 15px;">
    <br/>
   </span>
  </p>
  <p style="margin: 0cm 8px 0.0001pt;line-height: 20.4pt;background: white;">
   <span style="letter-spacing: 0.4pt;text-indent: 24.6pt;font-size: 15px;">
    特别是近十五年来，中国教育规模，尤其是高教规模极速扩招，这对学术原创性造成了巨大压力，二者之间严重失衡。
   </span>
  </p>
  <p style="margin: 0cm 8px 0.0001pt;line-height: 20.4pt;background: white;">
   <span style="letter-spacing: 0.4pt;text-indent: 24.6pt;font-size: 15px;">
    <br/>
   </span>
  </p>
  <p style="margin: 0cm 8px 0.0001pt;line-height: 20.4pt;background: white;">
   <span style="font-size: 15px;">
    <span style="font-size: 15px;letter-spacing: 0.4pt;text-indent: 24.6pt;">
     我们读研时，研究生规模很小，筛选很严，教授几乎是手把手地教，论文甚至是一句话一句话地“抠”，无论是选题、资料、观点、逻辑、论证等，都一点不能马虎；而且那时没有就业压力，研究生被当成宝贝，被抢着要。
    </span>
    <span style='font-size: 15px;font-family: -apple-system-font, BlinkMacSystemFont, "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;letter-spacing: 0.4pt;text-indent: 24.6pt;'>
     可现在学术生态早已不是那时的逻辑了。
    </span>
   </span>
  </p>
  <p style="margin: 0cm 8px 0.0001pt;line-height: 20.4pt;background: white;">
   <span style="font-size: 15px;">
    <span style='font-size: 15px;font-family: -apple-system-font, BlinkMacSystemFont, "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;letter-spacing: 0.4pt;text-indent: 24.6pt;'>
     <br/>
    </span>
   </span>
  </p>
  <p style="margin: 0cm 8px 0.0001pt;line-height: 20.4pt;background: white;">
   <span style='font-family: -apple-system-font, BlinkMacSystemFont, "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;letter-spacing: 0.4pt;text-indent: 24.6pt;font-size: 15px;'>
    因此，目前出现的较严重的学术失范，甚至造假行为，从大原则上看，与公共政策的选择不无关系，当然，从另一方面考虑，这种政策选择的缺陷，不能成为社会投机心态膨胀和个别人行为失当的解脱理由。
   </span>
  </p>
  <p style="margin: 0cm 0cm 0.0001pt;line-height: 20.4pt;background: white;">
   <br/>
  </p>
  <p style="white-space: normal;margin-left: 8px;margin-right: 8px;">
   <span style="color: rgb(0, 122, 170);font-size: 15px;">
    <strong>
     <span style="color: rgb(0, 122, 170);font-size: 15px;font-family: 宋体;">
     </span>
    </strong>
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="font-size: 15px;">
    <strong>
     <span style="font-size: 15px;font-family: 宋体;color: rgb(0, 122, 170);letter-spacing: 0.5px;">
      ◆ 学术规范和伦理知识多以短期讲座和专题教育的形式传授，这次“升级”为一门课程，区别在哪里？
     </span>
    </strong>
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="font-size: 15px;">
    <strong>
     <span style="font-size: 15px;font-family: 宋体;color: rgb(0, 122, 170);letter-spacing: 0.5px;">
      <br/>
     </span>
    </strong>
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="font-size: 15px;">
    <strong>
     <span style="font-size: 15px;font-family: 宋体;letter-spacing: 1px;">
      萧：一是
     </span>
    </strong>
    <span style="font-size: 15px;letter-spacing: 1px;">
     ，这是一种教学方式的改革，以往这类课程多为“
     <strong>
      <span style="font-size: 15px;font-family: 宋体;">
       规训
      </span>
     </strong>
     <strong>
      式
     </strong>
     ”的授课，教师总是摆出一副真理拥有者的架子，并以“应该”“必须”“一定要”等命令的修辞和口吻去教训人。这在学生的心理上会造成被当成“假想敌”的抵触情绪，效果不够理想。
    </span>
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="letter-spacing: 1px;font-size: 15px;">
    <br/>
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="font-size: 15px;">
    <strong>
     <span style="font-size: 15px;letter-spacing: 1px;">
      我们的教学改革，就是期望通过严肃的学术对话、平等讨论和师生互动的方式，把可能的问题摆出来，通过知识要素和道德品行的讨论去辨析问题，最终使师生共同去自觉自愿地遵守和践行学术规范与学术伦理的准则和美德。
     </span>
    </strong>
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="letter-spacing: 1px;font-size: 15px;">
    <br/>
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="font-size: 15px;">
    <strong>
     <span style="font-size: 15px;font-family: 宋体;letter-spacing: 1px;">
      二是
     </span>
    </strong>
    <span style="font-size: 15px;letter-spacing: 1px;">
     ，这门课是为学生从文史哲社政等不同学科的角度，准备的一场交叉学科的知识盛宴。只需2个学分，学生就可以亲身领略“师大大师”风采，聆听众多名师的教诲。我想如果我是一位学生，一定会受益匪浅。
    </span>
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="letter-spacing: 1px;font-size: 15px;">
    <br/>
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="font-size: 15px;">
    <strong>
     <span style="font-size: 15px;font-family: 宋体;letter-spacing: 1px;">
      三是
     </span>
    </strong>
    <span style="font-size: 15px;letter-spacing: 1px;">
     ，这门课不仅讲“理”，还会对中外学术规范与伦理的理论和事件，进行案例分析。通过具体的学术创作生活的细节，展示学术规范与伦理失当的教训。
    </span>
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="letter-spacing: 1px;font-size: 15px;">
    <br/>
   </span>
  </p>
  <p style="margin-left: 8px;margin-right: 8px;">
   <span style="font-size: 15px;">
    <strong>
     <span style="font-size: 15px;font-family: 宋体;letter-spacing: 1px;">
      四是
     </span>
    </strong>
    <span style="font-size: 15px;font-family: 宋体;letter-spacing: 1px;">
     ，考虑到青年人的特点，我们甚至大胆地提出授课地点能不能走出教室，是不是可以让学生们在“欣赏”和“玩”中实现教学计划？例如，在春暖花开的季节，带领同学们去郊外踏青，同时开讨论会，畅谈学习体会，展开不同观点的辩论，为青年学子营造宽松自由的讨论氛围。
    </span>
   </span>
  </p>
  <p style="white-space: normal;">
   <br/>
  </p>
  <p style="text-align: justify;margin-left: 8px;margin-right: 8px;">
   <span style="color: rgb(0, 122, 170);font-size: 15px;">
    <strong>
     <span style="color: rgb(0, 122, 170);font-size: 15px;font-family: 宋体;">
      <strong style="font-size: 15px;white-space: normal;">
       <span style="color: rgb(0, 122, 170);font-size: 15px;font-family: 宋体;">
       </span>
      </strong>
     </span>
    </strong>
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="font-size: 15px;">
    <strong>
     <span style="font-size: 15px;font-family: 宋体;color: rgb(0, 122, 170);letter-spacing: 0.5px;">
      ◆ 课程介绍里有很多名师授课，您在邀约过程中，大家达成什么共识吗？
     </span>
    </strong>
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="letter-spacing: 1px;font-size: 15px;">
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="font-size: 15px;">
    <strong>
     <span style="font-size: 15px;font-family: 宋体;letter-spacing: 1px;">
      萧：
     </span>
    </strong>
    <span style="font-size: 15px;letter-spacing: 1px;">
     华东师大的优秀学者“大咖”“大腕”非常多，学术功底非常雄厚。这门课程的开设，得到了他们的鼎力支持，这让作为这门课的责任教授的我非常感动！
    </span>
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="letter-spacing: 1px;font-size: 15px;">
    <br/>
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="letter-spacing: 1px;font-size: 15px;">
    比如，童世骏书记在百忙中抽出时间给予支持，他甚至还把自己的讲稿早早地就发给我“审阅”。且不说这对我是不是的“折煞”，起码对于像我这样的普通教师是一种鼓励。当然，我在公开的采访中说这些，可不是有意吹捧领导哈（笑）。
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="letter-spacing: 1px;font-size: 15px;">
    <br/>
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="letter-spacing: 1px;font-size: 15px;">
    还有一些知名教授，为了讲题反复斟酌，跟我一而再再而三地讨论，最后才定下来。每每经历到这些事情，我都为学生们感到幸运。
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <br/>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <strong style='font-size: 15px;font-family: -apple-system-font, BlinkMacSystemFont, "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;'>
    <span style="font-family: 宋体;color: rgb(0, 122, 170);letter-spacing: 0.5px;">
     ◆ 从课程内容看，这门课更像是学理探究课，作为责任教授，您对这门课有什么预期？
    </span>
   </strong>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="letter-spacing: 1px;font-size: 15px;">
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="font-size: 15px;">
    <strong>
     <span style="font-size: 15px;font-family: 宋体;letter-spacing: 1px;">
      萧：
     </span>
    </strong>
    <span style="font-size: 15px;letter-spacing: 1px;">
     对于一名学者或学生而言，学术规范是最基本的要求，学术交流必须建立在原创的基础上。
    </span>
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="letter-spacing: 1px;font-size: 15px;">
    <br/>
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="letter-spacing: 1px;font-size: 15px;">
    华东师大是有着深厚人文传统的著名高校，而且有着自身明显的传统和学统。用一句通俗的话说，华东师大“底蕴雄厚，温文尔雅，低调奢华”。我本人受益匪浅，一直在学习和践行这一良好的学风。当然，这不是说我喜欢“顺从的小绵羊”，学术上也要有足够的挑战精神。
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="letter-spacing: 1px;font-size: 15px;">
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="letter-spacing: 1px;font-size: 15px;">
    我们开设这门课程的目的之一，是希望学生们能够从进校开始就融入到学校的传统氛围之中，能够继承和发扬学校的优良传统，树立诚信、踏实和负责的良好学术风尚。
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="font-size: 15px;">
    <span style="font-size: 15px;letter-spacing: 1px;">
    </span>
    <span style='font-size: 15px;letter-spacing: 1px;text-indent: 32px;font-family: -apple-system-font, BlinkMacSystemFont, "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;'>
    </span>
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="font-size: 15px;">
    <strong>
     <span style="font-size: 15px;font-family: 宋体;color: rgb(0, 122, 170);letter-spacing: 0.5px;">
      ◆ 从前期推送看，有网友留言希望有网络共享课，是否有考虑呢？
     </span>
    </strong>
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="letter-spacing: 1px;font-size: 15px;">
   </span>
  </p>
  <p style="line-height: 27px;background: white;margin-left: 8px;margin-right: 8px;">
   <span style="font-size: 15px;">
    <strong>
     <span style="font-size: 15px;font-family: 宋体;letter-spacing: 1px;">
      萧
     </span>
    </strong>
    <span style="font-size: 15px;letter-spacing: 1px;">
     ：这是第一次开课，每一堂课会有录像。等授课教师对所讲的每讲内容有更加规范和系统的整理和补充之后，再考虑“慕课”也不迟。
    </span>
   </span>
