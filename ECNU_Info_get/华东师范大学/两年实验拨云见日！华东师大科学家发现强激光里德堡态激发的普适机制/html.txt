  <span style="font-size: 15px;">
   华东师范大学吴健教授团队利用飞秒强激光与分子相互作用产生里德堡原子，并结合电子-原子核关联能谱技术，历时两年实验，揭示了
   <strong>

   </strong>

     “ Electron-nuclearcorrelated multiphoton-route to Rydberg fragments of molecules ”
    </strong>

 <p style="white-space: normal;text-align: center;font-family: Optima-Regular, PingFangTC-light;line-height: normal;">
  <span style="font-size: 14px;color: rgb(136, 136, 136);">
   吴健教授（右）与博士研究生张文斌
  </span>
 </p>
 <p style="white-space: normal;">

 </p>
 <p style="white-space: normal;text-align: center;">
  <img class="" data-copyright="0" data-ratio="0.454" data-type="gif" data-w="1000" src="https://mmbiz.qpic.cn/mmbiz_png/bAZ0XOxmB0ppfcS6Hb4BZzVZNtkIsl2c9dZ1RoqiacxZTsggiafvxk2HryGmp22P8SXZyv4uZviaDic8gFInJ9xdnA/640?wx_fmt=gif"/>
 </p>
 <p style="white-space: normal;text-align: center;">
  <br/>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <span style="font-size: 15px;">
   <strong>
    不为人知的是，这项研究成果从最开始实验数据获得到最终文章发表历时将近2年。
   </strong>
  </span>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <br/>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <span style="font-size: 15px;">
   2017年初，研究团队发现强激光场作用下产生的中性里德堡原子和带电的粒子一样可以被探测器探测到。然而，他们分析数据发现中性里德堡原子的原子核能谱出现了奇怪的尖锐峰结构，这与之前研究人员提出的常规的受挫隧穿图像的预测结果很不一样。
  </span>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <br/>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <span style="font-size: 15px;">
   在这之后很长的一段时间里，研究团队进行了非常多次的讨论和分析，不断提出新的物理解释，但又很快被自己否定。
  </span>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <br/>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <span style="font-size: 15px;">
   但他们并没有因此放弃，而是不断提高测量的精度和分辨率，并测试不同的物理条件下里德堡态的激发过程，希望可以了解背后真正普适的物理机制。
  </span>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <br/>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <span style="font-size: 15px;">
   <strong>
    在经历了数不胜数的实验以及反复讨论后，吴健教授团队最终发现，当把电子与原子核关联起来考虑时，所有问题都迎刃而解了。
   </strong>
  </span>
  <br/>
 </p>
 <p style="white-space: normal;">
  <br/>
 </p>
 <p style="white-space: normal;">
  <br/>
 </p>
 <p style="white-space: normal;text-align: center;font-family: Optima-Regular, PingFangTC-light;">
  <img class="" data-copyright="0" data-ratio="0.6666666666666666" data-s="300,640" data-type="jpeg" data-w="1269" src="https://mmbiz.qpic.cn/mmbiz_jpg/bAZ0XOxmB0ppfcS6Hb4BZzVZNtkIsl2cib4l7ticrnsIib8QBPPiabtgqdqiaUfxGnc4qGUnG0g6uG4O0NY7R4l41iaQ/640?wx_fmt=jpeg"/>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <span style="font-size: 14px;color: rgb(136, 136, 136);">
   吴健教授课题组成员
  </span>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <br/>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <br/>
 </p>
 <p style="white-space: normal;text-align: center;">
  <img class="" data-copyright="0" data-ratio="0.376" data-type="gif" data-w="1000" src="https://mmbiz.qpic.cn/mmbiz_png/bAZ0XOxmB0ppfcS6Hb4BZzVZNtkIsl2cUZTZV8mlNmdhxSQIEgcbHK7u8eXfFQgLtVOXUZdrTibMs2UEamgO2Gg/640?wx_fmt=gif"/>
 </p>
 <p style="white-space: normal;text-align: center;">
  <br/>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <span style="font-size: 15px;">
   超快强激光作用下，原子或分子内的束缚电子将从光场中吸收光子能量发生电离。根据激光强度的不同，电子的超快电离可以理解为多光子电离或量子隧穿机制。近年来，研究人员发现，在强激光场作用下，电子有一定的概率不被电离而被囚禁在里德堡态，形成稳定的中性里德堡原子分子。
   <br/>
  </span>
 </p>
 <p style="white-space: normal;">
  <br/>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <span style="font-size: 15px;">
   中性里德堡原子分子具有诸多显著的特征，例如巨大的轨道半径、极长的相干时间、较大的碰撞截面、强电偶极矩以及高电极化率等。这些特性使得里德堡原子分子在精密光谱与精密测量、量子非线性光学、非平衡量子多体动力学、量子计算与量子信息等交叉研究前沿中有许多重要的应用。
  </span>
 </p>
 <p style="white-space: normal;">
  <br/>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <span style="font-size: 15px;">
   作为产生里德堡原子分子的重要手段之一，强激光诱导里德堡态激发在中性原子加速、近阈值谐波产生、低能光电子谱结构产生以及多光子拉比振荡等强场物理现象中同样有着重要的应用。
  </span>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <br/>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <span style="font-size: 15px;">
   经过不断的科学探索，研究人员提出强激光诱导里德堡态激发的物理机制与原子分子电离机制类似，可以用多光子共振激发或受挫量子隧穿图像来解释。多光子共振激发机制指出，电子吸收多个光子能量后直接共振布居到里德堡轨道上，而受挫量子隧穿图像指出部分隧穿电子由于从光场中获得的动能较小，不足以克服原子核库伦势的束缚，从而在激光场消失后有一定的概率会被原子核重新俘获，形成高激发的中性里德堡态原子。
  </span>
 </p>
 <p style="white-space: normal;">
  <br/>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <span style="font-size: 15px;">
   然而，自从强激光诱导里德堡态激发现象被发现以来，强激光场作用下产生里德堡原子分子的物理机制一直在多光子共振激发机制和受挫隧穿机制二者之间饱受争议。其主要原因是，在过去的研究中，里德堡原子分子激发过程中电子与原子核之间的关联效应一直被忽略。
  </span>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <span style="font-size: 16px;">
  </span>
 </p>
 <p style="white-space: normal;text-align: center;">
  <img class="" data-copyright="0" data-ratio="0.376" data-type="gif" data-w="1000" src="https://mmbiz.qpic.cn/mmbiz_png/bAZ0XOxmB0ppfcS6Hb4BZzVZNtkIsl2czlBPsIENoLE7RAibCQjWBUMo4yH1G1n2Bt2qW2pf7mCib6jF82ic8CyUA/640?wx_fmt=gif"/>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <span style="font-size: 15px;">
   <br/>
  </span>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <strong>
   <span style="font-size: 15px;">
    氢气分子（H2）作为最简单的两电子中性分子系统，在揭示分子基本动力学过程方面扮演了十分重要的角色。
   </span>
  </strong>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <br/>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <span style="font-size: 15px;">
   基于此前发展的中性里德堡原子探测技术，吴健教授团队提出利用紫外飞秒强激光脉冲与氢气分子相互作用，开展强激光诱导里德堡态激发过程的实验探索。
  </span>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <br/>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <span style="font-size: 15px;">
   <strong style="font-size: 16px;">
    通过符合探测光电离解离产生的离子H+、中性里德堡原子H*和自由电子，并结合电子-原子核关联能谱技术，实验揭示了多光子共振激发为强激光诱导里德堡态产生的普适机制。
   </strong>
  </span>
  <br/>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <br/>
 </p>
 <p style="white-space: normal;text-align: center;font-family: Optima-Regular, PingFangTC-light;line-height: normal;">
  <img class="" data-copyright="0" data-ratio="0.715702479338843" data-s="300,640" data-type="png" data-w="1210" src="https://mmbiz.qpic.cn/mmbiz_png/bAZ0XOxmB0ppfcS6Hb4BZzVZNtkIsl2crZSId7zJVNR98qPpJgMShMtcUqdSmUibCtCj9DBdBTFdiaicFnBgtPfcg/640?wx_fmt=png"/>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;line-height: normal;">
  <span style="font-size: 14px;color: rgb(136, 136, 136);">
   超短飞秒激光诱导氢气分子解离产生中性里德堡原子及其符合探测示意图
  </span>
 </p>
 <p style="white-space: normal;text-align: center;font-family: Optima-Regular, PingFangTC-light;line-height: normal;">
  <img class="" data-copyright="0" data-ratio="0.9886039886039886" data-s="300,640" data-type="png" data-w="1053" src="https://mmbiz.qpic.cn/mmbiz_png/bAZ0XOxmB0ppfcS6Hb4BZzVZNtkIsl2cIbfDuicsNMgE9iaandKrNlI7iceo0T0uwic2S6Y0DQLYHt1Kib5Hqlx2icicA/640?wx_fmt=png"/>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;line-height: normal;">
  <span style="font-size: 14px;color: rgb(136, 136, 136);">
   电子-核关联能谱揭示氢气分子双电离通道和里德堡态激发通道中电子与核之间能量如何分配
  </span>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <span style="font-size: 16px;">
  </span>
  <br/>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <span style="font-size: 15px;">
   利用电子-核关联能量谱，强激光场作用下氢气分子双电离通道和里德堡原子激发通道均可用三步过程很好的描述。
  </span>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <br/>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <span style="font-size: 15px;">
   首先，氢气分子发生单电离过程；其次，产生的氢气分子离子（H2+）核波包将经由不同的光子分辨路径在势能曲线上运动；最后，当核波包运动到一定的核间距时会进一步发生共振增强电离跃迁到库伦排斥态，最终解离形成两个质子,即双电离通道，或者经多光子共振激发后布居到高激发里德堡态，最终解离形成一个质子和一个中性里德堡原子，即里德堡原子激发通道。
  </span>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <br/>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <span style="font-size: 15px;">
   实验结果显示，里德堡态多光子共振激发时的核间距要小于发生共振增强电离时的核间距。另外，由于斯塔克位移效应的影响，发生里德堡态共振激发处的核间距大小随着激光强度的增加而变大。这一变化将影响电子与解离原子核之间的分配比，从而引起里德堡原子的能谱结构随光强的变化。当光强达到一定强度时，氢气分子双电离通道和里德堡原子激发通道的解离原子核能谱变得非常相似。
  </span>
 </p>
 <p style="white-space: normal;">
  <br/>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <span style="font-size: 15px;">
   这一现象表明，多光子共振激发机制作为强激光诱导里德堡态产生的普适机制，同样可以很好的解释受挫隧穿电离理论的预测结果。
   <strong>
    该项研究揭示了分子内电子-核关联效应在里德堡原子产生的过程中的重要性，极大深化了我们对强激光诱导里德堡态激发这一基本物理行为的认识，为强场里德堡原子分子激发的相干调控提供了新方法和新思路。
   </strong>
  </span>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <br/>
 </p>

 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <span style="font-size: 14px;color: rgb(136, 136, 136);">
   氢气分子双电离通道和里德堡原子激发通道的解离原子核能谱随激光强度的变化
  </span>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <span style="font-size: 16px;">
  </span>
  <br/>
 </p>
 <p style="white-space: normal;font-family: Optima-Regular, PingFangTC-light;">
  <span style="font-size: 15px;">
   吴健教授团队长期从事超快强激光作用下分子超快行为精密测量与控制方面的研究，例如围绕分子内电子-核关联效应，发展了电子-离子多体符合成像技术和电子-原子核关联能谱技术，首次实验观测到分子内电子-核在多光子吸收过程中的能量关联共享效应[Phys. Rev. Lett.111, 023002 (2013)]，揭示了分子振动态布居作为电子-核共享多光子能量的物理机制[Phys. Rev. Lett. 117,103002 (2016)]，利用电子重散射为原子核存储更多的光子能量提供了崭新的途径[PNAS 115, 2049 (2018)]。2017年，进一步发展了中性里德堡原子探测技术，首次实现了强激光场作用下分子内电子重俘获超快动态过程的精密测量与操控[Phys.Rev.Lett.119,253002(2017)]。
