  <span style="font-size: 16px;letter-spacing: 0.5px;">
  </span>
  <span style="letter-spacing: 0.5px;font-size: 14px;">
   黑白相间的苗条小鸟：白鹡鸰
  </span>
 </p>
 <p style="line-height: 1.5em;text-align: justify;margin-left: 8px;margin-right: 8px;">
  <span style="font-size: 16px;letter-spacing: 0.5px;">
   <br/>
  </span>

 <p style="line-height: 1.5em;text-align: justify;margin-left: 8px;margin-right: 8px;">
  <span style="font-size: 16px;letter-spacing: 0.5px;">
   丽娃河边不时会出现一些黑白相间的苗条小鸟——
   <strong>
    白鹡鸰
   </strong>
   ，它们喜欢在水边的草地漫步，有时候还会在丽娃河中的荷叶上踱步，不过更多的时候，我们会看到它们在空中呈现波浪状的飞行姿态。
  </span>
 </p>
 <p style="line-height: 1.5em;text-align: justify;margin-left: 8px;margin-right: 8px;">
  <span style="font-size: 16px;letter-spacing: 0.5px;">
   <br/>
  </span>

 <p style="text-align: center;margin-left: 8px;margin-right: 8px;">

 <p style="line-height: 1.5em;text-align: center;margin-left: 8px;margin-right: 8px;">

  <span style="letter-spacing: 0.5px;font-size: 14px;">
   夜鹭，是校园常见鸟类中体型最大的
  </span>

 <p style="line-height: 1.5em;text-align: justify;margin-left: 8px;margin-right: 8px;">

 <p style="line-height: 1.5em;text-align: justify;margin-left: 8px;margin-right: 8px;">
  <span style="font-size: 16px;letter-spacing: 0.5px;">
   继续沿着河边走，丽娃河中间的喷泉设施上偶尔会有一只蓝黑色一动不动的大鸟，这是夜鹭，是校园常见鸟类中体型最大的鸟。傍晚时分从丽虹桥上空略过的它们，会发出“嘎嘎嘎”的沙哑叫声。
  </span>

  <span style="letter-spacing: 0.5px;font-size: 14px;">
   会游泳的鸟类：黑水鸡
  </span>
 </p>
 <p style="line-height: 1.5em;text-align: justify;margin-left: 8px;margin-right: 8px;">
  <span style="font-size: 16px;letter-spacing: 0.5px;">
   <br/>
  </span>
 </p>
 <p style="line-height: 1.5em;text-align: justify;margin-left: 8px;margin-right: 8px;">
  <span style="font-size: 16px;letter-spacing: 0.5px;">
   丽娃河中也不缺乏会游泳的鸟类，它的羽毛全身黑色，名字叫做黑水鸡。我们只有很少的机会能够在夏雨岛北岸的芦竹和芦苇等挺水植物丛中发现它们隐秘的身影。
  </span>
 </p>
 <p style="line-height: 1.5em;text-align: justify;margin-left: 8px;margin-right: 8px;">

  <span style="letter-spacing: 0.5px;font-size: 14px;">
   十分娇小的小鸟：柳莺类
  </span>

 <p style="line-height: 1.5em;text-align: justify;margin-left: 8px;margin-right: 8px;">
  <span style="font-size: 16px;letter-spacing: 0.5px;">
   <br/>
  </span>
 </p>
 <p style="line-height: 1.5em;text-align: justify;margin-left: 8px;margin-right: 8px;">
  <span style="font-size: 16px;letter-spacing: 0.5px;">
   想必你一定听说过西湖十景之一的“柳浪闻莺”。这其中的“莺”指的就是一类十分娇小的小鸟——柳莺。
  </span>
 </p>
 <p style="line-height: 1.5em;text-align: justify;margin-left: 8px;margin-right: 8px;">
  <span style="font-size: 16px;letter-spacing: 0.5px;">
   <br/>
  </span>
 </p>
 <p style="line-height: 1.5em;text-align: justify;margin-left: 8px;margin-right: 8px;">
  <span style="font-size: 16px;letter-spacing: 0.5px;">
   在华东师大的丽娃河边，也时常能够听到小小的柳莺们“叽叽呖呖”的叫声，婉转动听。
  </span>
  <span style='font-family: -apple-system-font, BlinkMacSystemFont, "Helvetica Neue", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei", Arial, sans-serif;font-size: 16px;letter-spacing: 0.5px;'>

  </span>
 </p>
 <p style="line-height: 1.5em;text-align: center;">
  <br/>
 </p>
 <p style="line-height: 1.5em;text-align: justify;margin-left: 8px;margin-right: 8px;">
  <span style="font-size: 16px;letter-spacing: 0.5px;">
   体型略微大一些的暗绿绣眼鸟长着一副白眼圈，它们的叫声也同样悦耳，春天校园里的怒放的各种樱花丛中是最容易发现它们的地方。
  </span>

 <p style="line-height: 1.5em;text-align: justify;margin-left: 8px;margin-right: 8px;">

  </span>

 <p style="line-height: 1.5em;text-align: justify;margin-left: 8px;margin-right: 8px;">
  <span style="font-size: 16px;letter-spacing: 0.5px;">
   <br/>

 <p style="line-height: 1.5em;text-align: justify;margin-left: 8px;margin-right: 8px;">
  <span style="font-size: 16px;letter-spacing: 0.5px;">
   华东师大的两校区都有银杏林、水杉林，这里是师大学子晨读的好去处，也是鸟类喜欢的家园。
  </span>
 </p>
 <p style="line-height: 1.5em;text-align: justify;margin-left: 8px;margin-right: 8px;">

   <br/>

 <p style="line-height: 1.5em;text-align: justify;margin-left: 8px;margin-right: 8px;">
  <span style="font-size: 16px;letter-spacing: 0.5px;">
   树林下方的灌木丛中，隐藏着“校园里最萌的小鸟”——
   <strong>
    棕头鸦雀
   </strong>
   。这种小鸟也喜欢集成小群一起活动，不过它们活动的地点都是银杏林周围一圈茂密的珊瑚树树篱里面，所以我们通常只能听到“啾~啾~啾”的连贯叫声。棕头鸦雀那一副圆圆的小脑袋似乎直接安在圆滚滚的身体上，可谓萌劲十足。
  </span>
 </p>
 <p style="line-height: 1.5em;text-align: justify;margin-left: 8px;margin-right: 8px;">

 <p style="line-height: 1.5em;text-align: center;margin-left: 8px;margin-right: 8px;">

  </span>
