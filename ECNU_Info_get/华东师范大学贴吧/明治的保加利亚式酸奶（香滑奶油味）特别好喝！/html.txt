   不知道是不是大家的考试要到了，最近图书馆的小朋友也开始多了起来，让在一旁找书的鸡排也感觉年轻了不少，每次匆忙的找好书，然后匆忙的离开，总想做很多事，可能就会在无形中错过了更多的事。

 <p style="text-align: left;">
  <span style="font-size: 18px;">
   预定的新版《上海堡垒》前天到了，很幸运是签名本之一，和《哈利波特与被诅咒的孩子》一起收到，鸡排晚上回到宿舍后稍微熬了一下夜，尽管感冒加重了，也还是勉强读完了两本书，对于一个看过很多遍旧版《上海堡垒》的读者，还是更推荐略显青涩、欲语还休的旧版，不过新版其实也还好，没有那么不堪。

   附上一首诗给大家，叶芝的《当你年老时》，也是《上海堡垒》中一直提到的很可爱的一首诗。

  </span>
 </p>

  </span>
 </p>
 <p style=" margin-bottom: 16px; color: rgb(51, 51, 51); ;  font-size: 16px; white-space: normal; text-align: center; ">
  <span style="color: rgb(51, 51, 51); font-size: 18px;">
   当你老了，头白了，睡思昏沉；
  </span>
 </p>
 <p style=" margin-bottom: 16px; color: rgb(51, 51, 51); ;  font-size: 16px; white-space: normal; text-align: center; ">
  <span style="color: rgb(51, 51, 51); font-size: 18px;">
   炉火旁打盹，请取下这部诗歌；
  </span>
 </p>
 <p style=" margin-bottom: 16px; color: rgb(51, 51, 51); ;  font-size: 16px; white-space: normal; text-align: center; ">
  <span style="color: rgb(51, 51, 51); font-size: 18px;">
   慢慢读，回想你过去眼神的柔和；
  </span>
 </p>
 <p style=" margin-bottom: 16px; color: rgb(51, 51, 51); ;  font-size: 16px; white-space: normal; text-align: center; ">
  <span style="color: rgb(51, 51, 51); font-size: 18px;">
   回想它们昔日浓重的阴影；
  </span>
 </p>
 <p style=" margin-bottom: 16px; color: rgb(51, 51, 51); ;  font-size: 16px; white-space: normal; text-align: center; ">
  <span style="color: rgb(51, 51, 51); font-size: 18px;">
   多少人爱你青春欢畅的时辰；
  </span>
 </p>
 <p style=" margin-bottom: 16px; color: rgb(51, 51, 51); ;  font-size: 16px; white-space: normal; text-align: center; ">
  <span style="color: rgb(51, 51, 51); font-size: 18px;">
   爱慕你的美丽、假意或真心；
  </span>
 </p>
 <p style=" margin-bottom: 16px; color: rgb(51, 51, 51); ;  font-size: 16px; white-space: normal; text-align: center; ">
  <span style="color: rgb(51, 51, 51); font-size: 18px;">
   只有一个人爱你那朝圣者的灵魂；
  </span>
 </p>
 <p style=" margin-bottom: 16px; color: rgb(51, 51, 51); ;  font-size: 16px; white-space: normal; text-align: center; ">
  <span style="color: rgb(51, 51, 51); font-size: 18px;">
   爱你衰老了的脸上痛苦的皱纹；
  </span>
 </p>
 <p style=" margin-bottom: 16px; color: rgb(51, 51, 51); ;  font-size: 16px; white-space: normal; text-align: center; ">
  <span style="color: rgb(51, 51, 51); font-size: 18px;">
   垂下头来，在红光闪耀的炉子旁；
  </span>
 </p>
 <p style=" margin-bottom: 16px; color: rgb(51, 51, 51); ;  font-size: 16px; white-space: normal; text-align: center; ">
  <span style="color: rgb(51, 51, 51); font-size: 18px;">

 </p>
 <p style=" margin-bottom: 16px; color: rgb(51, 51, 51); ;  font-size: 16px; white-space: normal; text-align: center; ">

 </p>
