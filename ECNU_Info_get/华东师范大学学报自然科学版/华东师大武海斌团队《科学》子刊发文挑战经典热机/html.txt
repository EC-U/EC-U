 <p style="text-align: justify;line-height: 1.75em;">
  <span style="font-size: 15px;letter-spacing: 1px;">
   <br/>
  </span>
 </p>
 <p style="text-align: justify;line-height: 1.75em;">
  <span style="font-size: 15px;letter-spacing: 1px;">
   <span style="font-size:11.5pt;color:#3e3e3e;letter-spacing:.75pt;">
    <span style="font-family:微软雅黑;">
     发动机是现代工业的心脏
    </span>
    <span style="font-family:微软雅黑;">
     ，
    </span>
    <span style="font-family:微软雅黑;">
     与人们的生产生活息息相关
    </span>
    <span style="font-family:微软雅黑;">
     。
    </span>
    <span style="font-family:微软雅黑;">
     传统意义上的发动机是一种将其他形式的能量转化成机械能对外做功从而驱动某些器械的机器
    </span>
    <span style="font-family:微软雅黑;">
     ，
    </span>
    <span style="font-family:微软雅黑;">
     如内燃机
    </span>
    <span style="font-family:微软雅黑;">
     （
    </span>
    <span style="font-family:微软雅黑;">
     汽油发动机等
    </span>
    <span style="font-family:微软雅黑;">
     ）、
    </span>
    <span style="font-family:微软雅黑;">
     外燃机
    </span>
    <span style="font-family:微软雅黑;">
     （
    </span>
    <span style="font-family:微软雅黑;">
     斯特林发动机
    </span>
    <span style="font-family:微软雅黑;">
     、
    </span>
    <span style="font-family:微软雅黑;">
     蒸汽机等
    </span>
    <span style="font-family:微软雅黑;">
     ）、
    </span>
    <span style="font-family:微软雅黑;">
     电动机等
    </span>
    <span style="font-family:微软雅黑;">
     。
    </span>
   </span>
  </span>

 <p style="text-align: justify;line-height: 1.75em;">

   <br/>
  </span>
 </p>
 <p style="text-align: justify;line-height: 1.75em;">
  <span style="font-size: 15px;letter-spacing: 1px;">
   <span style="font-size:11.5pt;color:#3e3e3e;background:white;letter-spacing:.75pt;">
    <span style="font-family:微软雅黑;">
     但传统的发动机在效率和功率两者之间寻找平衡
    </span>
    <span style="font-family:微软雅黑;">
     ：
    </span>
    <span style="font-family:微软雅黑;">
     提高其中一个参数时
    </span>
    <span style="font-family:微软雅黑;">
     ，
    </span>
    <span style="font-family:微软雅黑;">
     另一个参数值必然要下降
    </span>
    <span style="font-family:微软雅黑;">
     ，
    </span>
    <span style="font-family:微软雅黑;">
     这主要是由于发动机在对外做功时所产生的摩擦耗散等所引起的
    </span>
    <span style="font-family:微软雅黑;">
     。
    </span>
    <span style="font-family:微软雅黑;">
     膨胀或者压缩过快使发动机工作介质不仅无法提高输出功率
    </span>
    <span style="font-family:微软雅黑;">
     ，
    </span>
    <span style="font-family:微软雅黑;">
     反而会产生极大的摩擦耗散进而降低输出效率
    </span>
    <span style="font-family:微软雅黑;">
     ，
    </span>
    <span style="font-family:微软雅黑;">
     严重影响其输出功率
    </span>
    <span style="font-family:微软雅黑;">
     ，
    </span>
   </span>
   <span style="font-size:11.5pt;font-weight:bold;color:#3e3e3e;background:white;letter-spacing:.75pt;">
    <span style="font-family:微软雅黑;">
     导致经典发动机的热机效率往往低于其理论极限
    </span>
    <span style="font-family:微软雅黑;">
     。
    </span>

  </span>
  <span style="letter-spacing: 1px;color: rgb(136, 136, 136);font-size: 14px;">
   量子热机绝热冲程的量子摩擦和对外做功
  </span>
 </p>
 <p style="text-align: justify;line-height: 1.5em;">
  <span style="letter-spacing: 0.5px;margin: 0px;line-height: 150%;font-family: 宋体;font-size: 9pt;">
  </span>
  <span style="letter-spacing: 1px;font-size: 14px;color: rgb(136, 136, 136);">
   A图表征绝热冲程的量子摩擦，局域透热驱动量子绝热捷径过程（红色曲线和蓝色数据）能够回归至1表征系统量子摩擦被大幅度抑制，实现了零摩擦做功。B图表征系统对外做功，采用局域透热驱动的量子绝热捷径过程红色曲线系统对外做功增加42.3%，实现了最大功率输出。
  </span>
  <span style="letter-spacing: 0.5px;margin: 0px;line-height: 150%;">
  </span>
 </p>
 <p style="text-align: center;">
  <span style="font-size: 14px;letter-spacing: 1px;">
   <br/>
  </span>
 </p>
 <p style="text-align: justify;line-height: 1.75em;">
  <span style="font-size: 15px;letter-spacing: 1px;">
  </span>
 </p>
 <p style="text-align: justify;line-height: 1.75em;">
  <span style="font-size: 15px;letter-spacing: 0.5px;">
   量子热机可突破当前热机输出功率和效率互相限制的难题，在相同能量输入条件下可输出更多的能量，并且速率更快。但量子绝热过程中不可避免产生的量子摩擦，制约着量子热机的效率和输出功率，且量子绝热过程所花费时间越短，量子摩擦越大。
  </span>
 </p>
 <p style="text-align: justify;line-height: 1.75em;">
  <br/>
  <span style="font-size: 15px;letter-spacing: 1px;">
  </span>
 </p>
 <p style="text-align: justify;line-height: 1.75em;">
  <span style="font-size: 15px;letter-spacing: 1px;">
  </span>
 </p>
 <p style="text-align: justify;line-height: 1.75em;">
  <span style="font-size: 15px;letter-spacing: 0.5px;">
   有别于经典热机的是，量子热机使用量子系统作为工作物质，其摩擦损耗可以通过量子绝热操控来得到有效抑制，甚至在一定条件下利用相干特性可以突破经典热机极限，制成高效率高输出功率的量子热机。
  </span>
 </p>
 <p style="text-align: justify;line-height: 1.75em;">
  <br/>

 <p style="text-align: right;line-height: 1.75em;">
  <span style="letter-spacing: 0.75pt;line-height: 1.75em;text-align: justify;font-size: 11.5pt;color: rgb(62, 62, 62);">
   <span style="font-family:微软雅黑;">
    <br/>
   </span>
  </span>
 </p>
 <p style="text-align: justify;line-height: 1.75em;">
  <span style="font-size: 15px;letter-spacing: 0.5px;">
   武海斌教授课题组利用强关联费米多体系统实现了量子绝热冲程的零摩擦做功，有效提高了输出效率和输出功率。在强关联量子多体系统中，每个原子都可以视为一个单独的量子热机，多体的强关联系统使量子热机在工作时相互协作、共同对外做功。
  </span>
  <br/>
 </p>
 <p style="white-space: normal;text-align: justify;line-height: 1.75em;">

  </span>
 </p>
 <p style="text-align: justify;line-height: 1.75em;">
  <span style="font-size: 15px;letter-spacing: 0.5px;">
   “这项研究所运用的透热驱动的量子绝热捷径方案，是一种无耗散的量子绝热驱动方案，很大程度上缩短整个量子绝热冲程所花费的时间，而且不会产生量子摩擦，实现高功率输出。”武海斌说。

  <span style="font-size: 15px;letter-spacing: 0.5px;">
   近年来，武海斌课题组在量子热力学和动力学研究中开展了一系列研究，实验研究了强相互作用超冷费米气体普适的Efimovian膨胀动力学[Science 353, 371(2016)]以及超级Efimovian膨胀动力学[Phys. Rev. Lett. 120, 125301 (2018)]；并且首次在强相互作用超冷费米气体中实现了有限时间内的量子绝热捷径[Phys. Rev. A 97, 013628 (2018)]，在强相互作用超冷量子气体的量子热力学操控中积累了丰富的研究经验。
